# Amazon商品データ収集ツール
指定したキーワードのAmazon検索結果（商品名、価格、URL）を自動収集し、分析可能なCSV形式で出力するPythonツールです。

## 機能
- スクレイピング対策（bot検知）を回避する処理を実装
- 取得データを数値型に変換（データクレンジング）
- エラー発生時のスキップ処理による安定稼働

## 使用技術
- Python
- Selenium (WebDriverWaitによる待機処理)
- Pandas (データ整形)

※このリポジトリは学習・検証目的のサンプル実装です。実際のサービス運用で利用する場合は、必ず対象サイトの利用規約・robots.txt・関連法令および公式APIの利用ポリシーを確認・遵守してください。
